Hi. First of all, I have to say that Cryopid is really cool. :) It 
actually works well, unlike many other similar software that I've tried.

I noticed that the command name of a frozen process (as shown by 'ps') 
is not preserved. For example, it I freeze 'cat' into 'cat.core' and run 
that, then the resulting process will have the command name 
'[cat.core]'. This happens even if I execlp() it with a different value 
for its argv[0].

I also noticed that resetting argv[0] in a resumed process doesn't work. 
If I do that then the command name becomes an empty string.

Is there a way to preserve the command name, or is there a way to set it?
