larger file version for mkstemp, mkstemp64.
As the stdlib.h header we use from the host machine the mkstemp is replaced
with mkstemp64 if *_FILE_OFFSET_BITS *is defined.
Some little large file support is only present dietlibc, glibc and uclibc
have the support for mkstemp64.
